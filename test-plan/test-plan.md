# Test Plan — TaskFlow

## Objective
Verify functionality, stability and usability of TaskFlow application.

## Test approach
- Manual black-box testing
- API validation via Postman
- UI checks across browsers

## Features under test
- Login / Registration
- Task CRUD operations
- User assignment
- Status updates

## Entry criteria
- Application is deployed and accessible
- Test environment is stable

## Exit criteria
- All critical bugs fixed or documented
- Core functionality tested successfully
